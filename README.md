## *Quresh Traders*

### *Description*
Quresh Traders is a Flask-based web application designed to streamline inventory workflows and automate data processing.
Key Features

    Automated Inventory: Developed a system to populate and manage product inventory by processing Excel files with Openpyxl.

    Smart Data Extraction: Integrated Adobe’s PDF Services API to extract structured data from PDF documents efficiently.

    Report Generation: Built a feature to generate PDF quotations directly from the SQL database using SQLAlchemy ORM.

### *Technologies*

    Backend: Python, Flask, SQLAlchemy

    Libraries/APIs: Openpyxl, Adobe PDF API, SQLite/PostgreSQL
